1. Select the folder where all the images are stored for cropping
2. Select the folder where all exported images will go, CAREFUL this can overwrite your originals if you let it!
3. Enter the Start X and Y (top left corner) either as pixel values, or check "As %" to make it a percentage (eg: 50 is 50%, would cut down the middle of a dual-monitor screenshot)
4. Enter the End X and Y (bottom right corner) in the same manner
5. GO! and crop your images, enjoy!
